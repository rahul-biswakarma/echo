# Echo

## Model Download Instructions

To use this application, you'll need to download the Whisper model:

1. Download the `ggml-small.bin` model from the [Hugging Face repository](https://huggingface.co/ggerganov/whisper.cpp/tree/main)
2. Direct download link: https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-small.bin
3. Place the downloaded model file in the `models` directory of the repo
